Glossary Of Terms

The glossary below covers key terms and technical jargon related to the JustSayin mobile application, which offers daily quotes based on user-selected categories.

  • API (Application Programming Interface)

    A set of protocols for building and integrating application software. In the context of JustSayin, APIs may be used to fetch quotes from external sources or synchronize user settings across devices.

  • Categories

    Predefined themes or topics under which quotes are organized in the JustSayin app. Users can select from categories like Humor, Love, Philosophy, etc., to receive customized daily quotes.

  • Device Compatibility

    Refers to the ability of the JustSayin app to function across different mobile devices and operating systems, ensuring a broad user reach.

  • Filter

    A feature that allows users to sort or display content based on specific criteria. In JustSayin, the filter is used in the 'Choose Categories' menu to display categories matching the user's input.

  • Notification Settings

    Controls within the app that allow users to configure when and how they receive notifications about new quotes or updates from the app.

  • Preferences

    User-defined settings that influence how the app behaves and what content it delivers. In JustSayin, preferences include selected categories for daily quotes.

  • Push Notifications

    Messages that pop up on a user's mobile device, sent by the app to alert the user about new quotes or important updates. These can be customized in the app's settings.

  • Search Functions

    A tool within the JustSayin app that allows users to quickly find specific categories by typing keywords into a search bar.

  • Settings

    An area in the JustSayin app where users can modify their app configuration, such as notification preferences and category selections.

  • Sync

    The process of ensuring that user data and preferences are consistent across various devices and sessions. This feature is crucial for users who access JustSayin from multiple devices.

  • UI (User Interface)

    The layout and design through which users interact with the JustSayin app. It includes buttons, text, images, and all other elements that users can see and interact with.

  • UX (User Experience)

    Refers to the overall experience of a user when interacting with the JustSayin app, especially in terms of usability, ease of use, and satisfaction.

  • Widget

    A small application or component that can be added to a user's home screen. JustSayin provides a widget that displays daily quotes directly on the home screen for easy access.

This glossary aims to clarify the technical terms associated with the JustSayin mobile application, enhancing the understanding and usage of the app for all users.
